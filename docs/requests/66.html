▮▮▮ THE CHICKEN AND EGG ▮▮▮

A convention that says "every PR ships its own HTML doc" can't satisfy itself for the PR that establishes it — the PR number doesn't exist until the PR opens. The resolution: open the PR with the seed corpus only, capture the assigned number (#66), then add this very page as a follow-up commit on the same branch before merge. The corpus stays complete; the rule applies from day one.
